# Times Tables

A simple rust program that shows the times tables of an integer.

## Usage

`times_tables n`

where n is a number

## Example

`times_tables 142`

will give the output

```
142     x       1       =       142
142     x       2       =       284
142     x       3       =       426
142     x       4       =       568
142     x       5       =       710
142     x       6       =       852
142     x       7       =       994
142     x       8       =       1136
142     x       9       =       1278
142     x       10      =       1420
```
